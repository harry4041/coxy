Pixel perfect

Pixel perfect is a web game where you try your luck at picking the winning pixel. To play you have to create an account then choose a pixel. There is only a limited amount so get in there quick before someone else snaps up your lucky number! At the end of the month a pixel is chosen at random and the person who owns that pixel wins.

Users are created and sign in using flask_auth along with SQLAlchemy. When they choose a pixel, that pixels coordinates are saved against their profile.

Technologies

Python flask, python auth, SQLAlchemy, HTML, CSS, JavaScript

Twitter-bots

The road I travel the most is closed for roughly 3 months of the year due to flooding, and it isn't announced anywhere that it's flooded. I also am not a fan of getting to my car in the morning and realising too late that the windscreen is frozen. Therefore I made a twitter-bot that tweets out every day at 5am telling me if I have to deal with these things.

Sadly these ceased to tweet in November as the Heroku package I was using started to charge, and it didn't seem financially worth it!

Bounce

Bounce is a mobile game where the player controls the bottom ball with their finger and triess to stop the bouncing ball from getting past. If it does then it's game over and time to start again.

Though a simple game up front, it utilises a firebase realtime database to check if a user has played from their device before, create a user tied to that device if not, and save/retrieve score data. Some of this is tied up to the Unity front end to display user top scores.

Technologies

Unity, C#, Firebase
